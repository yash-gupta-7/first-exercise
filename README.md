# Gandhinagar Weather and AQI Analysis

## Overview

This project is a Python program that performs basic statistical analysis on the **weather and air quality data of Gandhinagar for the last 10 days**.

The program calculates:

- Average Temperature
- Median Temperature
- Average Humidity
- Median Humidity
- Average AQI (Air Quality Index)
- Median AQI

The results are displayed on the screen and also saved in a text file.

---

## Objective

The objective of this project is to:

- Understand basic data analysis using Python
- Learn how to calculate average and median
- Work with real-world environmental data
- Store analysis results in a file

## Technologies Used

- Python 3

- File handling
